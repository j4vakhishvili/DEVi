ატრიბუტები

HTML ატრიბუტები იძლევა დამატებით ინფორმაციას HTML ელემენტების შესახებ.

HTML ატრიბუტები

  • ყველა HTML შესაძლოა ჰქონდეს ატრიბუტები
  • ატრიბუტები იძლევა დამატებით ინფორმაციას ელემენტების შესახებ
  • ატრიბუტები ყოველთვის მითითებულია საწყის ტეგში
  • ატრიბუტები როგორც წესი იწარმოება მსგავსად: სახელი="მნიშვნნელობა"

ატრიბუტი - href

<a> თეგი აწარმოებს ლინკებს.

‎‎‏‏‎ ‎href‏‏‎ ‎

ატრიბუტი წარმოადგენს URL-ს რომელიც გადადის მითითებულ ლინკზე.

<a href="https://www.devi.edu.ge/">შეუერთდი DEVi-ს</a>

ლინკებზე მეტის გასაგებათ ეწვიეთ ამ გვერდს



ატრიბუტი - src

<img> თეგი გამოიყენება ფოტოების HTML გვერდზე გამოსატანად.

‎‎‏‏‎ ‎src‏‏‎ ‎

ატრიბუტი აწვდის ადგილის რომელიც მითითებულია სურათის გამოსატანად.

<img src="image.png">

მხოლოდ ორი გზაა, რათა ლინკების მიწოდება მოხდეს

‎‎‏‏‎ ‎src‏‏‎ ‎

ატრიბუტის მეშვეობით:

1. ბმულის სახით - ლინკების მეშვეობით, რომლებიც მოწოდებულია რომელიმე სხვა ვებსაიტიდან. (მაგ.: src="https://www.devi.edu.ge/image.png/")

შენიშვნა: საიტის გარედან შემოტანილი ფოტოები შესაძლოა შეიცავდეს საავტორო უფლებებს, თუ თქვენ არ ფლობს საავტორო უფლებებს შესაძლოა შეგექმნათ კანონთან არსებული პრობლემები.
ამავდროულად გაითვალისწინეთ, რომ თქვენ არ შეგიძლიათ მათი კონტროლი, არსებული ფოტოები შესაძლოა მათი სერვერებიდან წაიშალოს ან შეიცვალოს.

2. შიდა ფაილის სახით - მსგავის ტიპის სურათები საიტთან ერთად გაეშვება ონლაინში, შესაბამისად სურათს არ სჭირდება საიტის მისამართი, იგი პირდაპირ თქვენი ფაილებიდან ინლინკება. მაგალითად: src="image.png", თუ კი ფოტო სხვა ფაილში გაქვთ ამისთვის დაგჭირდებათ ფაილის არჩევა. მაგალითად: src="assets/image.png"

შენიშვნა: საუკეთესო ვარიანტია როდესაც ფოტოების საიტზე განთავსება შიდა ფაილის სახით ხდება, ვინაიდან მსგავს სიტუაციაში, ფოტო არასდროს წაიშლება, მაშინაც კი, თუ საიტი შეიცვლის მისამართს.

  • src ატრიბუტები <img> ტეგში წარმოადგენს ფოტოს ადგილმდებარეობის მითითებას
