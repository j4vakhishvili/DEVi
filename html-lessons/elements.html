მშობელი HTML ელემენტები

HTML ელემენტი შესაძლოა მშობელი იყოს.(ეს ნიშნავს, რომ ელემენტი შესაძლოა შეიცავდეს სხვა ელემენტებს).

ყველა HTML დოკუმენტი შეიცავს მშობელ HTML ელემენტს.

მოცემული მაგალითი გულისხმობს, რომ შეიცავს ოთხ HTML ელემენტს (<html> <body> <h1> და <p>).

მაგალითი

<!DOCTYPE html> <html> <head> <title>გვერდის სათაური</title> </head> <body> <h1>ჩემი პირველი სათაური</h1> <p>ჩემი პირველი პარაგრაფი</p> </body> </html>

მაგალითის ახსნა

<html> ელემენტი საწყისი ელემენტია და მთელ HTML დოკუმენტს აწარმოებს.

მას აქვს გამხსნელი თეგი <html> და დამხურავი თეგი </html>.

მაშასადამე <html> თეგის შიგნით მდებარეობს <body> ელემენტი.

<body> <h1>ჩემი პირველი სათაური</h1> <p>ჩემი პირველი პარაგრაფი</p> </body>

<body> ელემენტი აწარმოებს დოკუმენტის ტანს.

მას აქვს გამხსნელი თეგი <body> და დამხურავი თეგი </body>.

მაშასადამე <html> თეგის შიგნით მდებარეობს ორი განსხვავებული ელემენტი <h1> და <p>.

<h1>ჩემი პირველი სათაური</h1> <p>ჩემი პირველი პარაგრაფი</p>

<h1> ელემენტი წარმოადგენს სათაურს.

მას აქვს გამხსნელი თეგი <h1> და დამხურავი თეგი </h1>.

<p> წარმოადგენს პარაგრაფს.

მას აქვს გამხსნელი თეგი <p> და დამხურავი თეგი </p>.

არასდროს გამოტოვო დამხურავი თეგი!

ზოგიერთ ელემენტს შესაძლოა მაინც გამოჰქონდეს სწორი გამოსახულება, მიუხედავად იმისა თუ არ დახურავთ ელემენტს:

<body> <p>ჩემი პირველი პარაგრაფი <p>ჩემი პირველი პარაგრაფი </body>

მაგრამ ეს არ ნიშნავს იმას, რომ მათი ასე დატოვება ოდესმე არ გამოიწვევს ერორებს ან გაფრთხილებებს!
