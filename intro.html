გაცნობა

HTML არის სტანდარტული მერქაფ ენა ვებსაიტების შესაქმნელად.

რა არის HTML?

  • HTML იშიფრება როგორც: Hyper Markup Language.
  • HTML არის ვებ-გვერდების შექმნისთვის განკუთვნილი სტანდარტული მარკირების ენა.
  • HTML აღწერს ვებ-გვერდის სტრუქტურას.
  • HTML მოიცავს ელემენტების მრავალ ჯგუფს.
  • HTML ელემენტები გადასცემენ ბრაუზერებს კონტენტის გამოსახვის საშუალებებს.

მარტივი HTML დოკუმენტი:

<!DOCTYPE html> <html> <head> <title>Page Title</title> </head> <body> <h1>My First Heading</h1> <p>My first paragraph.</p> </body> </html>

My First Heading

My first paragraph.

მაგალითის ახსნა:

  • <!DOCTYPE HTML> დეკლარაცია განსაზღვრავს რომ ეს დოკუმენტი არის HTML დოკუმენტი.
  • <html> ელემენტი არის HTML გვერდის ფესვური ელემენტი.
  • <head> ელემენტი მოიცავს მეტა ინფორმაციას HTML გვერდის შესახებ.
  • <title> ელემენტი აკონკრეტებს HTML გვერდის. სათაურს(რომელიც ნაჩვენებია ბრაუზერის სათაურის ბარში ან გვერდის ტაბში).
  • <body> ელემენტი განსაზღვარვს დოკუმენტის "სხეულს", და არის ყველანაირი ხილული კონტენტის კონტეინერი, როგორებიცაა headings, paragraphs, images, hyperlinks, tables, lists და ა.შ.
  • <h1> ელემენტი განსაზღვრავს ფართო სათაურს.
  • <p> ელემენტი განსაზღვრავს პარაგრაფს.
